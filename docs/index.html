Sample Financial Calculation (Python)

# Simple NPV calculator for projected cashflows
def npv(rate, cashflows):
    return sum(cf / ((1 + rate) ** t) for t, cf in enumerate(cashflows))

cashflows = [-100000, 30000, 40000, 50000, 60000]
print(f"NPV @ 8%: {npv(0.08, cashflows):,.2f}")

Key Financial Formulas

Net Present Value (NPV):

\[\mathrm{NPV} = \sum_{t=0}^{T} \frac{CF_t}{(1+r)^t}\]

Weighted Average Cost of Capital (WACC):

\[\mathrm{WACC} = \frac{E}{E+D}r_E + \frac{D}{E+D}r_D(1 - T_c)\]
